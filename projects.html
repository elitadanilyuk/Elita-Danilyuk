ColorCoded was a semester long group project. The project was a part of a Modern Web Applications (CS312) class at Colorado State University, during Spring '22. The goal of the project is to have a website where users can chose colors and color in coordinates. Then, when they print the page, it prints a blank grid with the color coordinates so people can 'color by number'.

TRNG is a private project from another semester long group project. The project was a deliverable for Software Engineering (CS314) at Colorado Stat University, during Fall '21. The mobile web app is a trip planner. It has a map leaflet which you can search or click locations to add to a trip and it calculates individual and total distances from the selected locations. This project is private because it is still in use at CSU, if you would like access or have questions about the project, please contact me.
